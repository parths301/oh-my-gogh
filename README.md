# Oh my Gogh!

An upcoming brand selling lifestyle products for artists and creative minds. This repository contains the source code for the "Coming Soon" interactive landing page.

> **Status: under construction.**

Visit the website at: [ohmygogh.com](https://ohmygogh.com)

## Planned Product Range
- **Art supplies** — paints, brushes, sketchbooks, and tools
- **Apparel** — t-shirts and wearables inspired by art and artists
- **Prints & posters** — curated and original artwork
- **Stationery** — notebooks, cards, and desk goods
- **More to come** — an evolving catalogue for the creative lifestyle

## Features
- Interactive procedural ASCII art landing page.
- Customizable physics and aesthetic glow effects.
- Dynamic responsive design for desktop and mobile.

## Technologies Used
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla, Canvas API)
